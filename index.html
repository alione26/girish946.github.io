Basic access authentication uses standard fields in the HTTP headers for providing user credentials. It is the simplest technique for enforcing access control to web resources. It does not require cookies or session identifier or login page. The specifications for basic access authentication are specified in RFC7617 For Golang httpauth package provides HTTP Basic Authentication middleware. Installation $ go get github.com/goji/httpauth Usage In this example I have used it with vanilla net/http.
